Appendix G Emacs and Haiku ¶

Haiku is a Unix-like operating system that originated as a re-implementation of the operating system BeOS.

This appendix describes the peculiarities of using Emacs built with the Application Kit, the windowing system indigenous to Haiku. The idiosyncrasies illustrated here do not apply to Emacs on Haiku built without windowing support, or configured with X11.
