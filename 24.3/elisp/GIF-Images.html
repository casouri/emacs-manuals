38.16.5 GIF Images

For GIF images, specify image type gif.

:index index

You can use :index to specify image number index from a GIF file that contains more than one image. If the GIF file doesn’t contain an image with the specified index, the image displays as a hollow box. GIF files with more than one image can be animated, see Animated Images.
