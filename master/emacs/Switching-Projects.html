28.2.2 Switching Projects

Commands that operate on project files (see Project File Commands) will conveniently prompt you for a project directory when no project is current. When you are inside a project but you want to operate on a different project, the command project-switch-project can be used.

This command prompts you to choose a directory among known project roots, and then displays the menu of available commands to operate on the chosen project. The variable project-switch-commands controls which commands are available in the menu, and by which keys they are invoked.
